# The Bell Project

<img width="1448" height="830" alt="img_1" src="https://github.com/user-attachments/assets/b69b338f-88fd-412e-a804-bba62fe62238" />
The Bell Project is a non-profit newsroom investigating torture in the American criminal justice system. No other media organization tracks these extreme abuses on a national scale. Our reporting will expose the extent of this issue and its underlying causes for the first time.\


We are developing a public archive of torture allegations, an investigative series, and educational community programming to inform the public about this critical problem. The Bell Project will launch in late 2026.
